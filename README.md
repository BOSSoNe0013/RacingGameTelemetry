Racing Game Telemetry
---

Display telemetry data from racing games like Dirt 3/Rally/4 and F1 2015/16/17

## 1 - Requirements

- Python 3.5+
- Qt Python binding like PyQt5, PyQt4, PySide2 or PySide

## 2 - Run

    python ./main.py
